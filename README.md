# kiros33 Homebrew Tap

Homebrew tap for [Nexa Markdown Viewer](https://github.com/kiros33/sosomlab-nexa-viewer).

## Install

```sh
brew install --cask kiros33/tap/nexa-markdown-viewer
```

## Upgrade

```sh
brew update
brew upgrade --cask nexa-markdown-viewer
```

## Uninstall

```sh
brew uninstall --cask nexa-markdown-viewer
```
